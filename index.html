Need help?

Here's some keyboard shortcuts:

  • SPACE / ↑ to jump
  • S to start the game
  • R to reload
  • 1-3 to load a specific map

Wanna make your own map? ⚑

You've played a few rounds and want to try creating a new map? Awesome! You can add a map via your browsers localStorage (here's a guide for Chrome; other browsers work similarly). Simply, set the map key-value pair. The value can be any string of unicode characters. See the list below for more information on the building blocks.

Hit C to clear the local map key-value pair.

Here's some building blocks (i.e., unicode characters) to get you started:

  •   whitespace for the map (i.e., en quad), trust me it's there
  • ▖ bottom block (i.e., quadrant lower left)
  • ▘ top block (i.e., quadrant upper left)
  • ⚑ goal flag, *optional, but a nice touch (i.e., black flag)

*Only top/bottom blocks cause collisions (i.e., need to be avoided while playing). All other unicode characters will not harm the player (yep, even ãbç ...)
